1 - Refreshing the Resume Template Page

  • When you refresh the Page/Resume Template, a confirmation Dialog Box will be shown.
  • For Chrome Browser, if you select Reload, the page will be refreshed. If you select Cancel, the page will not refresh.
  • For Mozilla Firefox, if yoou select Leave Page, the page will be refreshed. If you select Stay on Page, the page will not refresh.

2 - Editing the Contents of the Resume Template

  • Editing text in the Resume Template is very simple.
  • In order to edit the text contents, you simply have to click on the area that you want to edit, remove the default text by Backspace and then type in the text content that you want.

3 - Deleting the contents which are not needed

  • If you think that you do not need any particular content, from any section of the Resume Template, then you can easily delete it.
  • In order to delete any content from any section, you simply have to select the content by clicking on it and then press Backspace or Delete, the content will be deleted.

Back To Top



4 - Undo Any Changes

  • Suppose you want to Undo any changes, Right Click anywhere on the Template -> Select Undo. The changes made will be reverted back.
  • You can also use Shortcut Key for Undo.

NOTE : Undo option will be available only if you have made any changes in the template.

5 - Adding Extra Experience in Work Experience Section

In case you want to add an extra Work Experience:-

  • Select a particular Work Experience Section (starting from Designation Name) -> Right Click on Selected Section -> Select Copy.
  • Click between the Blank Space between 2 Experience Section -> Press Right Click -> Select Paste.
  • The entire section will be copied.

6 - Adding Extra Projects in Project Section

In case you want to add an extra Project in the Project Section:-

  • Select a particular Project Section (starting from Project Name) -> Right Click on Selected Section -> Select Copy.
  • Click between the Blank Space between 2 Project Section -> Press Right Click -> Select Paste.

7 - Changing the Progress Bar Values in Skills Section

The default value of Progress Bar is set to 5. In order to change the Progress Bar Value:-

  • Click on any Progress Bar you want. A Dialog Box will be displayed.
  • Enter any value between 1 to 10. Click on OK.
  • The changes will be applied based on the value entered.

9 - Saving the PDF using Chrome

In order to save the Resume Template in PDF format using Chrome:-

  • Click on 3 dots present in the top right corner (below the Close Button) -> Select Print. A new window with appear.
  • In the Destination Option, select Save as PDF.
  • In Pages option, select All.
  • In Pages per Sheet option, select 1.
  • In Margins option, select None.
  • In Options, select Background Graphics. Click Save.

10 - Saving the PDF using Mozilla Firefox

In order to save the Resume Template in PDF format using Chrome:-

  • Click on Menus present in the top right corner (below the Close Button) -> Select Print. A new window with appear.
  • Select Page Setup Option in the top left.
  • In Format and Options Tab, select Orientation as Portrait.
  • In the Text Box below Orientation, enter 65. It will set the size of the content of the Resume in %.
  • In Options, select Print Background(color and images).
  • In Margins & Header/Footer Tab, set all the margins to zero and set all the Headers and Footers Section to --blank--. Select OK.
  • Select Print -> Microsoft to PDF. Select Page Range to All -> Click Print.
  • A new Dialog Box will appear to Save the Document. Enter the desired Name and Save it.
